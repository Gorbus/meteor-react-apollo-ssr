# Meteor, React & Apollo SSR Boilerplate

## Server side rendering ready, using the following libraries:

- React 17.x
- React Router 5.x
- Apollo Client 3.x
- React helmet async 1.x
- Styled components 5.x

Build on top of the classical apollo skeleton for Meteor.
